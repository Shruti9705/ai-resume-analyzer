# AI Resume Analyzer

An AI-powered platform to analyze resumes, calculate ATS scores, and provide keyword recommendations.

## Features
- **ATS Score Prediction**: Estimates how well a resume matches a job description.
- **NLP Analysis**: Extracts skills and keywords using Spacy/NLTK.
- **Recommendations**: Suggests missing keywords to improve ranking.
- **Web Interface**: Simple and intuitive UI for file uploads.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Python (Flask / FastAPI)
- **NLP**: Spacy, NLTK, PyPDF2
